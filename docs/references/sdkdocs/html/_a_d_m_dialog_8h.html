Dialog creation option: (Mac OS only) Sends enter events (Return key and keypad Enter) to a modal dialog's tracker before activating the OK behavior, and prevents that behavior if the tracker returns true.

See ADMDialogSuite10::SetTrackProc(). By default, enter events cause text item trackers to commit their text and return true, so this option normally prevents the "OK" behavior when Enter is pressed within a text item. For modal dialogs.

typedef ADMBoolean ADMAPI(*) ADMDialogTrackProc(ADMDialogRef inDialog, ADMTrackerRef inTracker)

Prototype for an event tracking procedure associated with a custom dialog.

Your function can access the event from the tracking object, and respond as desired. Use ADMDialogSuite10::SetMask() to specify which events are sent to the tracker. You can call the default tracker from this function. If you do not call the default tracker from your custom tracker, call ADMTrackerSuite2::Abort() to exit the tracking loop.

You might use a custom tracker, for example, to check for a modifier key, save the modifier state, and then call the default tracker. The default tracker returns true if the dialog is to be notified. Your dialog notifier procedure would act on the saved modifier state.

Parameters:
inDialog The dialog object.
inTracker The tracking object.
Returns:
True to send a notification to the dialog when the mouse button is released, or to indicate that a keystroke event was handled. False to send no notification, or indicate that a keystroke event was not handled.
See also:
ADMDialogSuite10::SetTrackProc()

Enumeration Type Documentation

enum ADMDialogStyle

Types of dialogs, used in ADMDialogSuite10::Create() and ADMDialogSuite10::Modal().

The type controls the behavior and appearance, including which predefined items are automatically added.

You can modify the type with ADMDialogSuite10::SetDialogStyle()

Enumerator:
kADMModalDialogStyle  Modal dialog, disallows event detection in other windows while the dialog is shown.

Activates the application when invoked. Compare kADMNoAppActivateModalDialogStyle.

kADMAlertDialogStyle  Alert dialog.

A modal dialog used to display a text message, specified in the dialog's text property. It contains only predefined standard dialog items.

kADMFloatingDialogStyle  Floating, non-modal dialog, or palette.

can be moved by user, drag bar on top.

kADMTabbedFloatingDialogStyle  Palette with a table that can be moved by user, drag bar on top.

Tabbed palettes can be docked together. See ADMDialogGroupSuite6.

kADMResizingFloatingDialogStyle  Palette that allows resize, can be moved by user, drag bar on top.
kADMTabbedResizingFloatingDialogStyle  Palette with a tab that allows resize, can be moved by user, drag bar on top.

Tabbed palettes can be docked together. See ADMDialogGroupSuite6.

kADMPopupDialogStyle  Popup palette.

Cannot be moved by user.

kADMNoCloseFloatingDialogStyle  Palette that cannot be closed by user (toolbar).
kADMSystemAlertDialogStyle  System alert dialog.

A modal dialog used to display a text message, specified in the dialog's text property. It contains only predefined standard dialog items. This differs from kADMAlertDialogStyle in Windows in that it always has the system buttons regardless of the state of kADMModalDialogHasPaletteSystemControlsOption settings.

kADMPopupControlDialogStyle  A popup dialog containing a UI control, associated with a dialog item of type kADMPopupControlType.

See ADMItemSuite9::SetPopupDialog().

kADMResizingModalDialogStyle  Modal dialog that allows resize.
kADMLeftSidedFloatingDialogStyle  Left-side palette (drag bar is on left side instead of top).
kADMLeftSidedNoCloseFloatingDialogStyle  Left-side palette that cannot be closed by user (toolbar).
kADMNoTitleDockFloatingDialogStyle 
Deprecated:
Untitled palette. Deprecated, do not use.
kADMTabbedHierarchyFloatingDialogStyle  Palette with a tab that can contain tree controls.

Tabbed palettes can be docked together. See ADMDialogGroupSuite6.

kADMTabbedResizingHierarchyFloatingDialogStyle  Palette with a tab that can contain tree controls, and allows resize.

Tabbed palettes can be docked together. See ADMDialogGroupSuite6.

kADMResizingPopupPaletteDialogStyle  Palette with non-hierarchical popup menu, that allows resize.
kADMPopupPaletteDialogStyle  Palette with non-hierarchical popup menu.
kADMHierarchyPopupPaletteDialogStyle  Palette with hierarchical popup menu.
kADMResizingHierarchyPopupPaletteDialogStyle  Palette with hierarchical popup menu, that allows resize.
kADMNoAppActivateModalDialogStyle  A modal dialog that does not activate the application when invoked.
kADMHostDefinedDialogStyle  A platform-specific host-created dialog.
