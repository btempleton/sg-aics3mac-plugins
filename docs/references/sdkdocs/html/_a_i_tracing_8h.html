Enumerator:

kAITracingVisualizeRasterOriginal  Original.

Include the original raster input (before preprocessing).

kAITracingVisualizeRasterPreprocessed  Preprocessed.

Include the preprocessed image.

kAITracingVisualizeRasterTransparency  Transparency.

Include a transparent version of the original raster input (before preprocessing).

Vector modes for tracing output in AITracingVisualizeVectorType.

Enumerator:
kAITracingVisualizeVectorArtworkNone  No Artwork.
kAITracingVisualizeVectorArtwork  Artwork.

The full result, paths and fills, of the tracing.

kAITracingVisualizeVectorPaths  Paths.

A paths-only version of the art with black stroked paths.

kAITracingVisualizeVectorPathsAndTransparency  Paths and Transparency.

A transparent version of the art with black stroked paths overlayed.
