Detailed Description

This suite provides a means for creating off-screen images that can be displayed and manipulated with ADMDrawerSuite6.

  • Acquire this suite using SPBasicSuite::AcquireSuite() with the constants kADMImageSuite and kADMImageSuiteVersion2.

ADMDrawerRef ADMAPI(* ADMImageSuite2::BeginADMDrawer)(ADMImageRef inImage)

Opens a sequence of calls to ADMDrawerSuite6 functions that operation on an image.

Locks the image object so that the drawer's data pointer remains valid.When the sequence is complete, call EndADMDrawer() to unlock the object.
