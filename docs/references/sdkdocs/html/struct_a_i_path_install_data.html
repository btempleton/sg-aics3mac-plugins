Detailed Description

Installation data for AIPathInterpolateSuite::InsertInterpolants():.

Initialize the parent, paintOrder and refChild to describe where to install the first interpolated path.

  • If refChild is non-null, it must be a child of parent, and the paintOrder must be kPlaceBelow or kPlaceAbove.
  • If refChild is NULL, paintOrder must be kPlaceInsideOnTop or kPlaceInsideOnBottom.

AIBoolean AIPathInstallData::endOnBottom

For default interpolation (when the post-install callback is NULL); ignored otherwise.

  • When true, each path is linked as the sibling of the previous one, so that the last created path, the one most like the supplied end path, is on the bottom.
  • When false, the previous path becomes the sibling of the next one created, so that the last created path, the one most like the supplied end path, is on the top.
