00065  *      The version number of the Sweet Pea host interface. You can compare it to the
00066  *      version number that Sweet Pea was compiled with to determine that you are
00067  *      running the version you think you are. See SPVersion() below.

00069  *      This number consists of a hi word which is the major version number reflecting
