enum AIRasterResolution

Raster resolution values for creating a new document, set in AINewDocumentPreset::docRasterResolution.

Values are in pixels-per-inch. This becomes the Raster Effect Settings value for the document.

Enumerator:
kAIRasterResolutionScreen  Low or screen resolution, 72 PPI.

This is the default.

kAIRasterResolutionMedium  Medium resolution, 150 PPI.
kAIRasterResolutionHigh  High resolution, 300 PPI.
