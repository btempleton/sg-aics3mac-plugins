Detailed Description

This class encapsulates a search through text.

The methods allow you to find and replace text in a document using various kinds of character matching.

To find or replace all matches in the search scope, do the following:

  • Save the text object you are currently working with.
  • Retrieve the current search position markers with GetPreReplaceAllSettings() and save them.
  • Implement a loop through all text objects, using FindMatch(), FindNextMatch(), and ReplaceMatch() to find or replace all occurances of your match string.
    Keep track of how many matches or replacements were found, to report back to the user.
  • Restore the previous search position markers with RestorePreReplaceAllSettings()
  • Restore the working text object with SetSearchRange().
