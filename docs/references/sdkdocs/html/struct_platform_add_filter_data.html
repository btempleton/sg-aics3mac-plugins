Detailed Description

The menu category and display name for a plug-in filter.

Filters in the same category are added to a submenu of the Filter menu.

  • In Windows, either or both can have a keyboard shortcut. Precede the shortcut character with an ampersand (&). Take care that the shortcut key does not conflict with other menu items.
