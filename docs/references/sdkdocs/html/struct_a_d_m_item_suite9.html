Detailed Description

A dialog item is a control element in a dialog.

Use this suite to create, manipulate, and query dialog items. You can also create items from the dialog object, using ADMDialogSuite10::CreateItem() and ADMDialogSuite10::CreateNestedItem().

  • Acquire this suite using SPBasicSuite::AcquireSuite() with the constants kADMItemSuite and kADMItemSuiteVersion9.
A dialog item has a parent dialog and a parent window. It's appearance and behavior are controlled by its type (ADMItemType), its style (ADMItemStyle), and properties that you can set and access using these functions, including developer-defined initialization, termination, drawing, and notification procedures.

You can specify a focus order for the items in a dialog. This is the order in which contained elements get input focus when the user presses the Tab key. To set the initial focus order for a dialog:

  • Call ADMDialogSuite10::BeginAdjustingFocusOrder()
  • Make a sequence of calls to SetFirstFocus() and SetNextFocus()
  • When the sequence is finished, call ADMDialogSuite10::DoneAdjustingFocusOrder() The focus order does not take effect until this call.
This code sequence must not be nested. Finish specifying the order in one dialog before proceeding to the next one.

To rearrange the order of an existing focus list, use MoveFocusItem().
