Detailed Description

This suite provides functions that allocate and dispose of blocks of memory.

This memory management scheme is generally deprecated. Do not use it to allocate memory. It is still used, however, by the AIMatchingArtSuite, whose functions return a handle that must be freed by the caller using MdMemoryDisposeHandle().

  • Acquire this suite using SPBasicSuite::AcquireSuite() with the constants kAIMdMemorySuite and kAIMdMemoryVersion.
