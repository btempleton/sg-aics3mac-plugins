#include <AILiveEffect.h>

List of all members.

Detailed Description

Sent when the user selects an effect whose input type is kInputArtDynamic.

If the type computation depends on input art, the plug-in can examine the passed art object to decide on the type.

long AILiveEffectInputTypeMessage::typeMask

[out] The type of art that can be operated on by this effect with this parameter set, an AIStyleFilterPreferredInputArtType value.

  • If the input art is a type that this effect can filter, set to kAnyInputArt.
  • Otherwise, set to the type to which the art should be converted.
