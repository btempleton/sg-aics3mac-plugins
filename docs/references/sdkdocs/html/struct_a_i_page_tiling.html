Detailed Description

The page tiling structure that describes how artwork is printed on one or more pages.

See AIDocumentViewSuite::GetPageTiling(). Tiling operates on an art piece, which can be a set of art objects, the entire art board, or a crop box.

AIRealRect AIPageTiling::bounds

The bounds of the artboard, whose dimensions are configured when creating a new document or through document setup.

Page tiles may extend beyond these bounds, but they must intersect it.
