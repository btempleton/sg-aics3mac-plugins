AIBoolean AIFlatteningInfoLinkedEPSObject::interacts

True if the linked object interacts with transparency.

  • If it does, it must be parsed into objects for printing.
  • If it does not, it can be passed directly to a PostScript device for printing.

AIArtHandle AIFlatteningInfoLinkedEPSObject::object

The linked EPS object.

AIBoolean AIFlatteningInfoLinkedEPSObject::parseOK

True if the contents of the linked EPS can be parsed for printing.

For example, in version 11 or later, a linked DCS cannot be parsed to recover the original content.
