Enumeration Type Documentation

enum AIHTMLConversionOptions

Bit flags for options that control how conversion is performed by AIHTMLConversionSuite::GetArtAsHTMLText().

The first bit is for all convertible types. Other options can be defined by an individual converter, which would also define a UI and rules for setting those options.

Enumerator:
kAIHTMLNoOptions  Clear all option flags.
kAIHTMLTextLinkTextOption  Set the dynamic link between a slice and its AI art.

See AISlicingSuite::GetSliceAsHTMLText()
