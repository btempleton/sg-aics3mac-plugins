Enumeration Type Documentation

enum AIGeometryOrganizationType

Types of art objects that can be enumerated to AIGeometryOrganizationProcs.

These correspond to AIArtType values for objects.

Enumerator:
kAIGeometryOrganizeNull  Marks the end of an object enumeration.
kAIGeometryOrganizePath  A path object.
kAIGeometryOrganizeGroup  A group object.
kAIGeometryOrganizeCompound  A compound object.
kAIGeometryOrganizePlaced  A placed object.
kAIGeometryOrganizeRaster  A raster object.
kAIGeometryOrganizeText  A text frame object.

(As of AI11 the text organization callbacks are called for each text frame.)

kAIGeometryOrganizeTextPath  A text path object.

(As of AI11 this is called if the text frame has a confining path--that is, it is area text or text on a path).

kAIGeometryOrganizeTextLine  A text line object.

(Obsolete, not invoked by AI11 or later.)

kAIGeometryOrganizeClipGroup  A clipping group object.
kAIGeometryOrganizeMask  A mask object.
kAIGeometryOrganizeMesh  A mesh object.
kAIGeometryOrganizeSymbol  A symbol object.
kAIGeometryOrganizeForeignObject  A foreign object.

Note:
AI11 and later.
kAIGeometryOrganizeLegacyText  A legacy text object.

Note:
AI11 and later.
kAIGeometryOrganizeDummy  Internal.
