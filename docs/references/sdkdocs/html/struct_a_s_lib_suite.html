Detailed Description

This suite provides cover functions for standard C library string utilities.

For additional details, see ANSI C documentation.

  • Acquire this suite using SPBasicSuite::AcquireSuite() with the constants kASLibSuite and kASLibVersion.

ASAPI char*(* ASLibSuite::strncat)(char *, const char *, size_t)

Appends a substring of one string to another.

The terminating null character in dst is overwritten by the first character of src. The resulting string includes a null-character at the end. If the terminating null-character appears in src before num characters have been appended, the function appends the null character to dst and ends.

Parameters:
dst The destination string, null-terminated with enough space to contain the original characters and the number of characters to append.
src The null-terminated string containing characters to append.
num The maximum number of characters to append.
Returns:
The destination string.

ASAPI char*(* ASLibSuite::strncpy)(char *, const char *, size_t)

Copies characters from one string to another.

Copies the first num characters of src to dst. No null-character is implicitly appended to dst after copying, so dst may not be null-terminated if no null characters are copied from src. If num is greater than the length of src, dst is padded with zeros until num.

Parameters:
dst The destination string, with enough space to contain num characters.
src The null-terminated string containing characters to copy.
num The number of characters to copy.
Returns:
The destination string.
