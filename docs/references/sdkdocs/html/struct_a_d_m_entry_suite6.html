List of all members.

Detailed Description

Lists are associated with dialog items of types that provides a list of choices, including list boxes, popup lists, popup menus, spin-edit popups, and text-edit popups.

A list object, ADMListRef, contains entries, ADMEntryRef. The individual entry objects contain the entry appearance and state information.

You create entries by adding them to a list, using either ADMListSuite4::InsertEntry() or Create().

This suite allows you to access and manipulate individual entries.

Individual entries have default behavior, such as responding to a click by becoming selected or deselected. You can provide a set of entry callback procedures for the containing list, like those associated with a dialog or item, which affect the appearance and behavior of the contained entries. A list callback can modify the default behavior of an entry, which you access through the entry object using functions in this suite.

  • Acquire this suite using SPBasicSuite::AcquireSuite() with the constants kADMEntrySuite and kADMEntrySuiteVersion6.

Creates an entry timer, which allows you to schedule an action for the entry.

The action can make use of developer-defined data associated with the entry; see GetUserData().

Parameters:
inEntry The entry object.
inMilliseconds The delay, in milliseconds, from when this call is executed until the action is called.

inTimeProc The procedure that implements the scheduled action.
inAbortProc The procedure to call if the timer is canceled.
inOptions Not used; pass 0.
Returns:
The timer object.
