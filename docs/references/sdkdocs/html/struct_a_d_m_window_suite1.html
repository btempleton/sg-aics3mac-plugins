#include <ADMWindow.h>

List of all members.

Detailed Description

On Mac OS only: These functions allow you to manage the position of dialog windows in the platform window occlusion order.

  • Acquire this suite using SPBasicSuite::AcquireSuite() with the constants kADMWindowSuite and kADMWindowSuiteVersion.
