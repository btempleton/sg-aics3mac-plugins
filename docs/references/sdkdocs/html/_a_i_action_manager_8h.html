enum _ActionDialogStatus

Choices for whether a parameters dialog appears by default when an action is played back.

This status is visually represented by a dialog icon to the left of the event name in the Actions palette.

Enumerator:
kDialogNone  No dialog is defined for this action.
kDialogOn  Dialog appears on playback.
kDialogPartialOn  Internal, not used by 3rd-party plug-ins.
kDialogOff  Dialog does not appear on playback.

kShowInPalette  shown
