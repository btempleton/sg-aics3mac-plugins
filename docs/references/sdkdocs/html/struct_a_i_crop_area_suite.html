Detailed Description

This suite allows you to activate, access, and modify crop areas in Illustrator documents.

Illustrator maintains a list of crop areas in a document, and these functions allow you to iterate through the list. The active crop area is the one returned by AIDocumentSuite::GetDocumentCropBox(), which is displayed with crop marks, and which is used when saving or exporting to a file.

  • Acquire this suite using SPBasicSuite::AcquireSuite() with the constants kAICropAreaSuite and kAICropAreaSuiteVersion.

AIAPI AIErr(* AICropAreaSuite::Next)(AICropAreaPtr *properties)

Finds the next crop area in the list after the currently active area, makes it active, and retrieves its properties.

The list is circular; that is, the next one after the last crop area is the first one in the list.

Parameters:
properties [out] A buffer in which to return the crop-area properties, or NULL if there are no crop areas in the current document.

AIAPI AIErr(* AICropAreaSuite::Previous)(AICropAreaPtr *properties)

Finds the previous crop area in the list before the currently active area, makes it active, and retrieves its properties.

The list is circular; that is, the previous one before the first crop area is the last one in the list.

Parameters:
properties [out] A buffer in which to return the crop-area properties, or NULL if there are no crop areas in the current document.
