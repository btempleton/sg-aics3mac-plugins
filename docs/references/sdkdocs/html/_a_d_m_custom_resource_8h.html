typedef ADMErr ADMAPI ADMAllocateBlockProc(ADMInt32 inSize, void **outBlock)

Prototype for a memory allocation callback procedure for a custom resource loader.

A composite resource parser can either create the object, or use this to return a block of data to be handled by another parser.

Parameters:
inSize The number of bytes to allocate.
outBlock [out] A buffer in which to return a pointer to the allocate block.
Returns:
An ADM error, kNoErr on success.
