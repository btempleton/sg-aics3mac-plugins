Detailed Description

This suite provides image-processing functions that preprocess images for use with tracing, preparing raster data for vectorization.

The image is manipulated in place.

  • Acquire this suite using SPBasicSuite::AcquireSuite() with the constants kAITracingIPSuite and kAITracingIPVersion.
