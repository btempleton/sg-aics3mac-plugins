Detailed Description

This suite provides a platform-independent interface to picture resources in Mac OS and Windows, which are represented as ADM icons.

  • Acquire this suite using SPBasicSuite::AcquireSuite() with the constants kADMIconSuite and kADMIconSuiteVersion2.

ADMIconRef ADMAPI(* ADMIconSuite2::Create)(ADMIconType inIconType, ADMInt32 inWidth, ADMInt32 inHeight, void *inData)

Creates an ADM icon using supplied image data.

The image data is not copied; if you release the memory, or if the image is in static memory that becomes invalid when the plug-in is unloaded, you must also destroy the icon object using Destroy().

Parameters:
inIconType The icon type constant.
inWidth The icon width in pixels. Families of icons should all be the same size.
inHeight The icon height in pixels.
inData A pointer to the image data.
Returns:
An ADM icon object.

ADMIconRef ADMAPI(* ADMIconSuite2::CreateFromImage)(ADMImageRef inImage)

Converts an ADM image to an ADM icon.

The ADM icon takes ownership of the image, so the caller does not need to deallocate it.

Parameters:
inImage The ADM image object; see ADMImageSuite2.
Returns:
The ADM icon object.

void ADMAPI(* ADMIconSuite2::Destroy)(ADMIconRef inIcon)

Removes an ADM icon object from memory.

If it was created using GetFromResource(), frees the image data. If it was created using Create(), you must free the image memory before calling this function.

Parameters:
inIcon The ADM icon object.
Returns:
Nothing.

ADMAPI void*(* ADMIconSuite2::GetData)(ADMIconRef inIcon)

Retrieves the image data reference by an ADM icon.

If you need the actual pixel information, you must parse the referenced data according to the platform resource specification or use platform-specific functions.

Parameters:
inIcon The ADM icon object.
Returns:
A pointer to the image data.

ADMIconRef ADMAPI(* ADMIconSuite2::GetFromResource)(SPPluginRef inPluginRef, const char *inName, ADMInt32 inIconID, ADMInt32 inIconIndex)

Creates an ADM icon using a resource loaded from a plug-in.

Searches the plug-in's resources for a PICT or BMP, and if not found, for an IconSuite or Icon. Does not load resources of type CICN.

Parameters:
inPluginRef The plug-in reference.
inName The unique, identifying name of the icon.
inIconID The platform-specific resource identifier.
inIconIndex A 1-based index added to the ID to get the final resource identifier of the icon.
Returns:
An ADM icon object. The type is kUnknown if a resource could not be loaded.
See also:
GetType()
