Dialog group position flag: True if the docked dialog is visible, false if it is hidden.

For a dialog to actually be shown, its own visibility must be true, it must be the front tab in its tab group, and floating dialogs must not be toggled off, as with the Tab key or ADMDialogGroupSuite6::ToggleAllFloatingDialogs().
