typedef ADMBoolean ADMAPI(*) ADMEntryTrackProc(ADMEntryRef inEntry, ADMTrackerRef inTracker)

Prototype for an entry event tracking procedure associated with a list.

Use ADMListSuite4::SetMask() to specify which high-level events are sent to the tracker. You can call the entry's default tracker from this function, using ADMEntrySuite6::DefaultTrack(). The default tracker handles entry selection, including multiple selection.

If you do not call the default tracker from your custom tracker, call ADMTrackerSuite2::Abort() to exit the tracking loop.

Parameters:
inEntry The entry object.
inTracker The tracker object.
Returns:
True to send a notification to the entry when the mouse button is released, or for edit-text items, to indicate that a keystroke was handled. False to send no notification, or indicate that the keystroke was not handled.
See also:
ADMListSuite4::SetTrackProc(), ADMEntrySuite6::DefaultTrack()
