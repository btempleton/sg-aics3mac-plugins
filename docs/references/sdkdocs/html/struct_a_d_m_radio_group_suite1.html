Detailed Description

This suite provides functions for managing radio button groups.

The buttons in a group are mutually exclusive. Selecting any button deselects all other buttons in the group.

  • Acquire this suite using SPBasicSuite::AcquireSuite() with the constants kADMRadioGroupSuite and kADMRadioGroupSuiteVersion1.
