Detailed Description

This suite provides utilities specific to the Mac OS platform.

  • Acquire this suite using SPBasicSuite::AcquireSuite() with the constants kAIMacUserSuite and kAIMacUserSuiteVersion.

AIAPI AIBoolean(* AIMacUserSuite::ModalFilterProc)(AIWindowRef theDialog, AIEvent *theEvent, short *itemHit)

A callback procedure that displays a platform-specific modal dialog in response to a user-interaction event.

(Note that this function returns a boolean value, not an error code.)

Parameters:
theDialog The window object for the dialog.
theEvent A pointer to the event object for the event that triggered the callback.
itemHit [out] A buffer in which to return the dialog result.
Returns:
True if the dialog was dismissed with the OK button, false if it was canceled.

AIAPI AIErr(* AIMacUserSuite::PreviewColorSwatch)(AIWindowRef window, AIRect *swatchRect, AIColor *colorSpec)

Displays a color swatch preview in a platform window, similar to the one shown in the Swatch palette.

Parameters:
window The window object.
swatchRect A pointer to the region in which to display the preview, in coordinates relative to the window's origin.
colorSpec A pointer to the swatch color.
