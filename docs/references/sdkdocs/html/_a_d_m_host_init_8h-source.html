00196 //      The host of ADM provides this suite for ADM to use at initialization.
00197 //      These values can also be set/reset later if desired using the ADM Host suites.

00199 //      Most suites must be fully initialized. Since this suite is intended to be

00201 //      any of these function pointers can be NULL and ADM will simply use the default

00203 //      directory on Windows and the host options. They are checked once at startup
00204 //      and never again.
