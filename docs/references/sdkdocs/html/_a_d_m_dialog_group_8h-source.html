00050 // -----------------------------------------------------------------------------
00051 //      For the xxGroupInfo() functions, positionCode is a code to restore a
00052 //      dialog's position within a docked/tabbed group. The group is
00053 //      referred to by name, this being the name of the ADM Dialog that is
00054 //      the first tab in the top dock of the group.
00055 //
00056 //      You don't need to know what positionCode means, but if you are curious:
00057 //
00058 //      byte                            meaning
00059 //      ----                            ------------------------------
00060 //        1 (0x000000ff)        bits 0-7, dock position, 0 is no dock, 1 is first docked (i.e. top dock), 2 etc.
00061 //        2 (0x0000ff00)        bits 8-15, tab position, 0 is no tab group, 1 is the 1st tab, 2 etc.
00062 //        3 (0x00010000)        bit 16, boolean, 1 for front tab.
00063 //        3 (0x00020000)        bit 17, boolean, 0 is zoom up, 1 is zoom down/collapsed.
00064 //        3 (0x00040000)        bit 18, boolean, 0 is in hidden dock, 1 is in visible dock.
00065 //      3-4 (0x00780000)        4 bits 19-22, pane index, 0 is no pane, 1 is the 1st pane counting from the center of the screen, etc.
00066 //      3-4 (0x01800000)        2 bits 23-24, pane location, see enum ADMPaneLocation
00067 //        4 (0x06000000)        2 bits 25-26, pane state, see enum ADMPaneState
00068 //        4 (0x08000000)        bit 27, 1 for palette drawer open in iconic pane
00069 //        4 (0x10000000)        bit 28, boolean, tab/icon hidden in parent: 1 for hidden, 0 for visible
00070 //        4 (0x00000000)        reserved, currently unused
00071 //
00072 //      So for stand alone palette, the tab, dock position code is (0, ?, 0, 0)
00073
