Detailed Description

Deprecated:
Use AIMenuSuite instead. This suite provides an older mechanism for manipulating Illustrator menu commands, It is provided for compatibility, but has been superseded by AIMenuSuite.

AIAPI AIErr(* AICommandManagerSuite::GetCommandIDFromShortCutKey)(int key, unsigned long admModifiers, AICommandID *commandID)

Retrieves a command identifier from the keyboard shortcut.

Parameters:
key The keyboard shortcut key code. See ADM documentation.
admModifiers A logical OR of modifier key flags, such as kADMMenuKeyDownModifier. See ADM documentation.
commandID [out] A buffer in which to return the command ID.
Returns:
The error kBadParameterErr if no command ID exists for this shortcut.
