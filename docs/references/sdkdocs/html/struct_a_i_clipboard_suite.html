Detailed Description

The clipboard suite enables plug-ins to register new clipboard format handlers.

The handler's main entry point will then receive messages requesting it to copy data to and from the clipboard as needed. The messages have caller kCallerAIClipboard. The possible selectors are:

  • kSelectorAIGoClipboard
  • kSelectorAICanCopyClipboard
  • kSelectorAICloneClipboard
  • kSelectorAIDisposeClipboard
Acquire this suite using SPBasicSuite::AcquireSuite() with the constants kAIClipboardSuite and kAIClipboardVersion.
