enum ADMIconType

Icon types.

Used as input when creating icons, or as output to determine the type of icons retrieved from resources. See #ADMIconSuite2::GetFromResource().

Enumerator:
kCICN  Mac OS 'CICN' icon type.
kPICT  Mac OS 'PICT' icon type.
kIconSuite  Mac OS Icon Suite icon type.
kWinIcon  Windows icon type.
kBMP  Windows BMP icon type.
kADMImageIcon  Platform-independent icon type.
kCGImage  Mac OS X CG image icon type.
kUnknown  Unknown icon type.

An icon of this type is created when you call ADMIconSuite2::GetFromResource() if the specified resource is not found.
