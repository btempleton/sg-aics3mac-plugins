enum ADMWindowUIMode

The screen mode for the document window.

See also:
ADMBasicSuite10::SetWindowUIMode(), ADMBasicSuite10::GetWindowUIMode()
Enumerator:
kADMUIMode_Freeform  The document window is a user-resizable window within the application window.

This corresponds to "Standard Screen Mode" in the Tools palette.

kADMUIMode_FullScreen  The document window fills the screen, with the application menu at the top but no title bar.

This corresponds to "Full Screen Mode with Menu Bar" in the Tools palette.

kADMUIMode_FullScreenNoMenu  The document window fills the screen, with neither the application menu nor a title bar.

This corresponds to "Full Screen Mode" in the Tools palette.

kADMUIMode_Maximized  The document window fills the screen, has no title bar, and cannot be resized vertically.

This corresponds to "Maximized Screen Mode" in the Tools palette.
