ADMFoundResourceData Struct Reference

#include <ADMCustomResource.h>

List of all members.

Detailed Description

A composite resource parser can either create the object, or return a block of data to be handled by another parser.

This structure, returned in response to the kADMCustomResourceCaller, contains the new object or block, and the associated resource data.
