Member Data Documentation

char* PlatformAddFileFormatExData::extension

A comma-delimited list of DOS-style extensions that are filtered in the Open dialog, or appended to the file name in the Saves As dialog.

For example, "ai,eps".

The list has a maximum of 31 characters, and each extension has a maximum of 5 characters.
