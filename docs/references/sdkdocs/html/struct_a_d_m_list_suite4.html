Detailed Description

List objects are associated with dialog items of types that provide a list of choices, including list boxes, popup lists, popup menus, spin-edit popups, and text-edit popups.

A list, ADMListRef, contains entries, ADMEntryRef. The individual entry objects contain the entry appearance and state information; see ADMEntrySuite6.

This suite allows you to create and manipulate a set of entries that make up a list, and iterate through them. It also provides access to the platform menu resource that implements the list.

The default behavior of a list is controlled at the item level, and additional behavior is associated with individual entries. You can provide a set of callback procedures for a list, like those associated with a dialog or item, which affect the appearance and behavior of the contained entries. A list callback can modify or replace the default behavior of the entries.

  • Acquire this suite using SPBasicSuite::AcquireSuite() with the constants kADMListSuite and kADMListSuiteVersion4.
