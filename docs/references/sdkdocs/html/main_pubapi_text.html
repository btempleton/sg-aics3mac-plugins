Adobe Text Engine (ATE)

Adobe Text Engine (ATE) function suites and wrapper classes provide an interface to text in Illustrator documents.

Note:
Do not use the C functions defined in ATESuites.h directly. Use the C++ wrapper classes in the ATE namespace instead.
