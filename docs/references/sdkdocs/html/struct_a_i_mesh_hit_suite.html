Detailed Description

This suite provides functions that operate on hit objects obtained by AIHitTestSuite functions, when the result of the test is a hit on a mesh object.

They allow you to identify the part of the mesh object that was hit, and to change the selection or iterate through elements of the mesh based on hit results.

  • Acquire this suite using SPBasicSuite::AcquireSuite() with the constants kAIMeshHitSuite and kAIMeshHitVersion.
