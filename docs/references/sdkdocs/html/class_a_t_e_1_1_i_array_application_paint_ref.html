ATE::IArrayApplicationPaintRef

ATE::IArrayApplicationPaintRef Class Reference

#include <IText.h>

List of all members.

Detailed Description

Provides access to an ordered collection of application paint objects, which encapsulate Illustrator-specific painting of fill and stroke for text, as defined by the AIATEPaintSuite.
