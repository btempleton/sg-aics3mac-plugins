Detailed Description

This suite allows you to access and manipulate the Flash movie player which is embedded in the application.

The Flash player is represented by an ADMItemRef of type kADMFlashPlayerType.

  • Acquire this suite using SPBasicSuite::AcquireSuite() with the constants kADMFlashPlayerSuite and kADMFlashPlayerSuiteVersion.

ADMErr ADMAPI(* ADMFlashPlayerSuite::CallMovieFunction)(ADMItemRef inItem, const ADMUnicode *inRequest, ADMUnicode **outResponse)

Calls an ActionScript function that is registered for a Flash player, using the ExternalInterface.addCallback method in the movie being played.

This function is the equivalent of FPP_CallFunction2 in the Flash External Player API.

Parameters:
inItem The item object, of type kADMFlashPlayerType, for the Flash player.
inRequest An XML element that specifies the function to invoke and any arguments.
outResponse A buffer in which to return the XML element response from the movie. May not be NULL. The caller assumes ownership of the returned string buffer, and must free it using SPBasicSuite::FreeBlock().
Returns:
An error code, kNoErr on success.

ADMErr ADMAPI(* ADMFlashPlayerSuite::GetADMFlashPlayerProcs)(ADMFlashPlayerProcs *outProcs)

Fills out the internal ADM Flash Player procedures table.

The entries are needed by ScriptUI (the JavaScript user-interface module) to support the FlashPlayer element. The contents of the returned function table are considered private to ADM.

Parameters:
outProcs [out] A buffer in which to return the internal ADM Flash Player procedure table.
Returns:
An error code, kNoErr on success.

ADMErr ADMAPI(* ADMFlashPlayerSuite::LoadMovie)(ADMItemRef inItem, const ADMUnicode *inURL, ADMBoolean inPlayerPush)

Loads a movie into the Flash player.

This function is the equivalent of FPP_LoadMovie in the Flash External Player API.

Parameters:
inItem The item object, of type kADMFlashPlayerType, for the Flash player.
inURL The URL path to the movie to be played. A UTF-16 encoded string formed according to the rules defined for the path argument to FPP_LoadMovie.
inPlayerPush True to load the movief from the specified path, false to load the movie after this call using PushDataBuf().
Returns:
An error code, kNoErr on success.

ADMErr ADMAPI(* ADMFlashPlayerSuite::PlayMovie)(ADMItemRef inItem, ADMBoolean inRewind)

Resumes playing the main timeline of the movie in a Flash player.

This function is the equivalent of FPP_Play in the Flash External Player API.

Parameters:
inItem The item object, of type kADMFlashPlayerType, for the Flash player.
inRewind When true, the movie plays from the beginning. When false, it plays from the point at which it was stopped.
Returns:
An error code, kNoErr on success.

ADMErr ADMAPI(* ADMFlashPlayerSuite::PushDataBuf)(ADMItemRef inItem, const void *inBuffer, ADMUInt32 inBufferSize)

Streams a movie's bits into a Flash player.

Use after passing false for the inPlayerPush argument to LoadMovie().

This function is the equivalent of FPP_PushDataBuf in the Flash External Player API.

Parameters:
inItem The item object, of type kADMFlashPlayerType, for the Flash player.
inBuffer A pointer to a memory buffer holding the movie bits to be pushed.
inBufferSize The number of bytes from the buffer to be pushed to the player.
Returns:
An error code, kNoErr on success.

ADMErr ADMAPI(* ADMFlashPlayerSuite::StopMovie)(ADMItemRef inItem)

Stops playing the main timeline of the movie in a Flash player.

This function is the equivalent of FPP_StopPlay in the Flash External Player API.

Parameters:
inItem The item object, of type kADMFlashPlayerType, for the Flash player.
Returns:
An error code, kNoErr on success.

ADMErr ADMAPI(* ADMFlashPlayerSuite::UpdateCallHostFunctionProc)(ADMItemRef inItem, ADMFlashHostCallbackProc inCallback)

Sets up the function to be invoked when a movie invokes an application function for a Flash player, via the ExternalInterface.call ActionScript function.

By default, the Flash player does not call any function when the movie invokes an application function.

The application function callback is the equivalent of FPA_CallFunction in the Flash External Player API.

See also:
ADMFlashHostCallbackProc
Parameters:
inItem The item object, of type kADMFlashPlayerType, for the Flash player.
inCallback The function to be called when the item's movie invokes an application function. Pass NULL to indicate that no function should be called when a movie invokes an application function.
Returns:
An error code, kNoErr on success.
